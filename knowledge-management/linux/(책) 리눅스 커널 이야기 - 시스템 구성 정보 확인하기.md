
시스템 문제점 분석 및 확인을 위해 시스템 구성 정보 확인하는 방법을 배워보자

# 커널 정보 확인하기


```
uname -a

uname -r
```


커널 디버그 메세지 확인
- 커널 부팅할 때 나오는 메시지와 운영 중에 발생하는 메세지를 볼 수 있음
- 구체적으로 확인할 수 있는 내용 
	- 커널이 메모리를 인식하는 과정
	- 커널이 하드웨어를 인식하고 드라이버를 올리는 과정
	- 부팅 시 적용된 커널 파라미터
```
dmesg
```
